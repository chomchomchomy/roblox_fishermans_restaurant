# 🎣 Project: Fisherman's Restaurant (Version 1.0)
**最終目標：** 累計収益 1億ドル（約150億円）の達成
**コア・コンセプト：** 「中毒性のある収集（Fisch風）」×「自動化された経営（My Restaurant風）」×「血統と育成（ダビスタ・たまごっち風）」の究極融合。

---

## 🛠 コアループ (The Core Loop)
1. **[Hunt]** 海や川で多種多様な魚を釣る（レア度・天候・時間による変動あり）。
2. **[Cook]** 釣った魚をキッチンへ持ち込み、料理を作成する。
3. **[Serve]** お客さんに提供し、ゴールドと評判（XP）を獲得。
4. **[Grow]** 稼いだ金で「より深い海へ行ける船」「強力な竿」「豪華な内装」を購入し、規模を拡大する。

---

## 📅 5段階の開発ロードマップ

### Phase 1: 孤独な漁師と海辺の屋台（サバイバル期）
**「まずは一本の竿と、小さな焚き火から」**
- [ ] **釣り**: 桟橋からの手釣り。アジやサバなどの一般的な魚がターゲット。
- [ ] **経営**: 釣った魚をその場で焼いて出す「海辺の屋台（2席のみ）」。
- [ ] **目標**: 最初の「まともな釣り竿」と「調理コンロ」の資金調達。
- **★Point**: 釣りの手応え（振動や音）にこだわり、最初の1匹の感動を最大化する。

### Phase 2: 出航とシーフード・ワゴン（拡大期）
**「世界が広がる、行動範囲の拡大」**
- [ ] **釣り**: 「ボロい小舟」を入手。少し沖に出てタイやヒラメを狙える。
- [ ] **経営**: 屋台から「キッチンカー（ワゴン）」へ進化。客数と単価が上昇。
- [ ] **目標**: 「中古の漁船」と「店舗用の土地」の頭金を貯める。
- **★Point**: 「自分で船を操縦する」という自由度を与え、探索の楽しさを教える。

### Phase 3: 直営レストランとプロ船員（プロフェッショナル期）
**「経営者としての第一歩」**
- [ ] **釣り**: 本格的な漁船。自動で魚を獲る「船員（たまごっち・BSS要素）」を雇用。
- [ ] **経営**: 固定店舗「シーフード・ダイナー」オープン。内装カスタマイズ解禁。
- [ ] **目標**: 伝説の巨大魚（クラーケン等）を釣るための特殊装備の入手。
- **★Point**: 自分が釣りをしている間も、店員が店を回す「放置収益」の快感を導入。

### Phase 4: 養殖プラントと高級ブランド（産業期）
**「血統と品質の支配」**
- [ ] **釣り**: 深海・極地へ遠征。超レア個体の捕獲。
- [ ] **養殖（ダビスタ要素）**: 釣った魚を「生け簀」で配合。世界に一匹の「ブランド魚」を創出。
- [ ] **経営**: 予約制の高級レストラン。産地・血統に応じた超高単価メニューの実装。
- **★Point**: 「配合の奥深さ」を入れ、プレイヤーを数千時間熱中させる沼を作る。

### Phase 5: 海の帝国とグローバル経済（ミリオン・ビリオン期）
**「1億ドルへの到達」**
- [ ] **Antigravity連携**: 全サーバー共通の「魚価マーケット」と「売上ランキング」。
- [ ] **ワールドレイド**: 巨大クジラや海の魔物を全プレイヤーで追うメガイベント。
- [ ] **UGC展開**: 自分の店の人気キャラや制服をRoblox公式アイテムとして販売。
- **★Point**: 150億円を支える「インフラ」として、外部サーバーによる徹底的なデータ管理。

## 🌟 差別化・収益化ポイント
- **鮮度ボーナス**: 釣った魚を店内の巨大水槽に飾り、客の満足度を上げる。
- **伝説の釣り場**: 特定の条件（雨の日、深夜など）にしか現れない幻のスポット。
- **釣り竿・ルアーガチャ**: 伝説の獲物を狙うための強力かつ美しい装備品。
- **水槽ディスプレイ**: 釣った魚を店内の巨大水槽に飾り、客の満足度を上げる。
